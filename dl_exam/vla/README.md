# :rocket: Model Introduction :rocket:

The autonomous driving VLA model evolved from VLM, with the Transformer/GPT architecture as its core. It realizes the unified modeling of vision-language-action through an explicit action decoder, integrates multi-modal sensor data and language instructions, and adopts a multi-stage training paradigm of "large-scale pre-training - multi-modal alignment - targeted enhancement - model compression". Combined with supervised imitation learning and reinforcement learning optimization, it possesses chain-of-thought reasoning ability to improve interpretability and generalization ability in long-tail scenarios. Currently, the main VLA models under development include the unified end-to-end VLA large models such as OpenDriveVLA, AutoVLA, IRL-VLA, CoReVLA, SimLingo, as well as inference-enhanced VLA models like AutoDrive-R2, Drive-R1, DSDrive, MTRDrive, etc. For the specific development of VLA technology in autonomous driving, please refer to [the latest vla model](https://mp.weixin.qq.com/s/vyetdRyVXJRifonRDM4BWQ) and [the history of vla model](https://mp.weixin.qq.com/s/fYj4N-kIOt0R-p_QkDkrDg).
